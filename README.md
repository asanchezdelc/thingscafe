# Things Cafe

Simple and easy device connection and dashboard for IoT. 

## Requirements
- PostgreSQL
- Redis
- NodeJS
- NPM

## Build from source
```
git clone https://github.com/asanchezdelc/thingscafe
cd thingscafe
npm install
npm start
```
